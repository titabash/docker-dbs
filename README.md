# docker-dbs

## Install
1. Install docker, docker-compose

## How to build
1. Enter the project root<br>
   ```cd <project directory>```
2. Build docker<br>
   ```make start```
3. Stop docker<br>
   ```make stop```
4. Remove docker<br>
   ```make remove```


## View logs
```docker logs <Container name> -f```
